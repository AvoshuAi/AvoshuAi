## Current Projects

### AI & Neurodivergent Tools

* 🧠 **[Arcus](https://github.com/avoshu/arcus)** - a self-hosted local AI voice assistant built for neurodivergent people and ADHDers who need a system that actually executes. not another chatbot. your own Jarvis, running locally, doing the work.

* 📊 **[LifeOS](https://github.com/avoshu/lifeos)** - an all-in-one personal dashboard for overthinkers. track your finances, habits, and life stats at a glance so you stop second-guessing and start knowing. built to eventually plug into Arcus.

* 🎬 **[StudioStage](https://github.com/avoshu/studiostage)** - a creative suite built for creators who do everything. thumbnails, graphics, branding assets, visual content - all in one place. stop hopping between tools.

### Web & UI

* 🌐 **[avoshu.com](https://avoshu.com)** - my personal portfolio and public landing page. the main place to find me on the internet.

* 🏢 **[voshustudios.com](https://voshustudios.com)** - Voshu Studios, my NJ-based creative agency. web design, branding, AI systems, and automation for local businesses.

* 🧪 **[voshulabs.com](https://voshulabs.com)** - Voshu Labs, the new home for all my apps, software, and tools. everything I build eventually lives here.

* 🤖 **[arcus.ai](https://arcus.ai)** - the public face of Arcus. a neurodivergent-first AI system built to compete with the big players, running on open source foundations with heavy custom code.
